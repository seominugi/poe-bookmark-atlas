# 자수정 글래스 (Amethyst Glass) — 디자인 시스템

북마크 아틀라스 / 아틀라스 제품군용 디자인 시스템. seominugi.com 글래스모피즘을
계승하되, **자수정(보라) 정체성 + 두꺼운 베벨 글래스 + 주기적 글린트**로 재정의했습니다.
성능 최우선: 무거운 움직이는 그라디언트("오로라")는 폐기하고 GPU 컴포지터 전용
효과(transform 글린트, 정적 글래스)만 사용합니다.

## 3. 표면 · 효과

### 글래스 패널
```
background: rgba(24,21,42,.93);
backdrop-filter: blur(8px) saturate(135%);   /* 성능: 가벼운 블러 + 낮은 불투명도 */
border: 1px solid rgba(255,255,255,.12);
border-top: 2px solid rgba(167,139,250,.55); /* 시그니처 자수정 상단 */
border-radius: 14px; overflow: hidden;        /* 스크롤 콘텐츠 클립 */
```

### 두꺼운 베벨 글래스 카드 (검색 다시 열기 = 클릭 영역)
```
background: linear-gradient(180deg, rgba(255,255,255,.08), rgba(167,139,250,.03)), rgba(20,18,34,.55);
border: 2px solid rgba(255,255,255,.18);
border-top-color: rgba(255,255,255,.42);     /* 상단 하이라이트 */
border-left-color: rgba(255,255,255,.30);    /* 좌측 하이라이트 */
box-shadow: 0 8px 28px rgba(0,0,0,.4),
            inset 0 1px 1px rgba(255,255,255,.24),   /* 안쪽 상단광 */
            inset 0 -1px 1px rgba(0,0,0,.22);         /* 안쪽 바닥음영 */
/* hover: 상단 테두리 자수정 + 보라 글로우 + translateY(-2px) */
```

### 글린트 (오로라 대체 — GPU 전용)
```
@keyframes ba-glint { 0%{transform:translateX(-220%) skewX(-14deg)} 55%,100%{transform:translateX(360%) skewX(-14deg)} }
/* 흰색 45% 폭 스트립이 주기적으로 버튼을 가로지름. 버튼별 duration/delay를 달리해 비동기 */
/* 아이템 시세 4.4s(-1s), 시장 동향 6.7s(-3.5s) */
```

## 4. 컴포넌트

- **버튼(의미색)**: 평상시 은은한 톤(색 soft 배경 + line 테두리), hover 시 진해짐. 정사각 아이콘 버튼은 29×29(카드 액션)/24×24(폴더).
- **카드/행**: 두꺼운 베벨 글래스. 북마크는 **이름 칩만** 클릭(재검색), 히스토리는 전체 클릭.
- **칩**: 조건 개수(`조건 N개`, hover 툴팁으로 그룹별 상세), 스탯 칩, 폴더 선택 칩.
- **폴더**: 글래스 카드로 감싸 영역 구분. 헤더에 +저장/위·아래 순서/이름변경/삭제/폴더 내보내기.
- **토스트**: 상단 중앙, 자수정 좌측 보더. 진입(`ba-toast-in`)·퇴장(`ba-toast-out`) 트랜지션.
- **툴팁**: 커스텀, 호버 즉시 표시. 조건 상세는 그룹별 들여쓰기(`white-space:pre`).
- **입력바**: 인라인 이름 입력 + 폴더 선택 칩(+ 새 폴더).
- **빈 상태**: 마스코트(애널리스트) + 안내. 북마크 0·폴더 0일 때만.
- **핸들**: 우측, 그립(드래그=ns-resize 위치 이동) / 라벨(클릭=접기). 접힘 시 북마크 카운트 배지.
- **가이드 투어**: 스포트라이트 코치마크. 실제 요소 측정 → 주변 디밍 + 자수정 링 + 말풍선. 스크롤 밖 타겟은 패널 안으로 이동.

## 5. 모션 · 접근성 규칙

- **모션**: 글린트(주기적 빛 반사)만 사용. 움직이는 그라디언트("오로라") 금지.
  카드 hover lift(-2px), 토스트 in/out, 링 이동 트랜지션.
- **성능**: 애니메이션은 `transform`/`opacity`만(컴포지터). backdrop-filter는 패널 1곳, 8px.
- **접근성**: 전역 `:focus-visible` 자수정 링, 주요 인터랙션 `aria-label`/`role="button"`,
  키보드(Tab→Enter/Space) 지원, `prefers-reduced-motion` 시 애니메이션·글린트 정지.
- **타깃 크기**: 카드 액션 버튼 24–29px, 핸들 30×116, 연령대 고려한 본문 하한.
