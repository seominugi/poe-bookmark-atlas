# 북마크 아틀라스 — 버전 원장 (CHANGELOG)

버전 스냅샷 + 핸드오프를 **한 곳에서** 묶는 기준 문서입니다.
디자인이 바뀌면 → ① 디자인 시스템 먼저 수정 → ② 산출물 반영 → ③ 이 문서 + `handoff/HANDOFF.md` 갱신, 순서를 지킵니다.

---

## 스냅샷 규칙 (왜 루트에 두는가)

`.dc.html`은 `./support.js` · `assets/` · 형제 `Icon.dc.html`을 **상대경로**로 참조합니다.
하위 폴더(`versions/…`)로 옮기면 이 경로가 깨져 미리보기가 뜨지 않으므로,
**스냅샷은 프로젝트 루트에 복제본으로** 둡니다.

- 큰 방향 전환 직전에 복제: `북마크 아틀라스.dc.html` → `북마크 아틀라스 v1.dc.html`
- 현재 활성 파일은 항상 접미사 없는 이름(`북마크 아틀라스.dc.html`)을 최신으로 유지
- 동결된 버전은 이름에 버전을 박아 보존(읽기 전용처럼 취급, 수정 금지)

## 산출물 ↔ 역할

| 파일 | 역할 | 비고 |
|---|---|---|
| `디자인 시스템.dc.html` | 📐 기준 문서 (SSOT) | 색·타이포·컴포넌트 변경은 여기 먼저 |
| `북마크 아틀라스.dc.html` | 🎯 메인 산출물 (인패널) | 실제 개발 이관 핵심 |
| `확장 팝업.dc.html` | 🎯 보조 산출물 (팝업) | 소개·단축키·가이드 재실행 |
| `Icon.dc.html` | 🔧 공용 부품 | 위 파일들이 `dc-import`로 사용 — 삭제 금지 |
| `handoff/` | 📦 개발 이관 패키지 | `HANDOFF.md` + 드롭인 `panel.css` + 마스코트 |

## v1.0 — 2026-06-23 (현재 동결 후보)

테마: **자수정 글래스 (Amethyst Glass)** — 보라 글래스 + 두꺼운 베벨 카드 + GPU 글린트.
성능 최우선: `backdrop-filter` 전면 미사용(불투명 배경), transform·opacity 전용 애니메이션.

### 기능 범위
- **자동 히스토리** — 거래소 검색 자동 적재
- **북마크/폴더** — 드래그 정리, 폴더 색상 구분(5색), 미분류 폴더
- **빠른 검색 + 정렬** — 북마크/히스토리 각각 이름·조건 검색, 정렬(순서/최근/이름) 분절 토글
- **리그 태깅 + 불일치 경고** — 저장 당시 리그 기록, 현재 리그와 다르면 "이전 리그" 배지 → 원클릭 재검색
- **조건 기반 이름 자동 제안** — 저장 시 검색 조건에서 핵심 스탯으로 이름 자동 채움
- **soft-stale** — 오래된 북마크는 삭제 대신 흐림 + "갱신 필요" 배지 → 원클릭 되살리기
- **중복 저장 점프** — 같은 조건 재저장 시 기존 카드로 이동 + 플래시
- **정보 밀도 토글** — 여유(노안 배려, 기본) / 조밀
- **대용량 리스트 대응** — 전 카드 `content-visibility:auto` + 히스토리 60개 캡 + "더 보기" 200개씩 + 검색
- **시세 버튼** — 아이템 시세 / 시장 동향 (마스코트 + 글린트), 서미누기 POE 경제 연결
- **피드백 소셜** — 패널 하단 고정(네이버 카페·유튜브·디스코드) + "서미누기 제작" 칩
- **가이드 투어** — 첫 실행 코치마크, 팝업에서 재실행
